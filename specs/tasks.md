# 実装計画: ブックマーク管理API

## 概要

FastAPI + SQLite によるブックマーク管理REST APIの実装タスクを整理する。

## タスク

- [x] 1. プロジェクト基盤のセットアップ
  - `api/` ディレクトリ構成を作成する
  - DB接続、テーブル初期化、アプリ起動処理を実装する
  - _要件: 6.1, 6.2, 6.3_

- [x] 2. Pydanticスキーマの定義
  - リクエスト/レスポンス用スキーマを実装する
  - _要件: 1.1, 1.6, 2.1, 3.1, 4.1, 5.1_

- [x] 3. フォルダ機能の実装
  - CRUD と削除時の参照解除を実装する
  - _要件: 2.1-2.6_

- [x] 4. タグ機能の実装
  - CRUD と重複制御を実装する
  - _要件: 3.1-3.7_

- [x] 5. ブックマーク機能の実装
  - CRUD、検索、絞り込み、ページングを実装する
  - _要件: 1.1-1.9_

- [x] 6. ブックマークへのタグ付与・解除
  - 付与と解除、重複付与エラーを実装する
  - _要件: 4.1-4.4_

- [x] 7. E2Eおよびプロパティテストの整備
  - API・フロントエンド・E2Eの観点を整理する
  - _要件: 全体_

## 備考

- 仕様変更時は `requirements.md` を先に更新する
- 実装差分が出た場合は `design.md` と `tasks.md` を追従させる
