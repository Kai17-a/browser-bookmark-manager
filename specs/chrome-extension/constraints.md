# 制約

- 拡張機能は Manifest V3 として実装する
- 保存対象は現在アクティブなタブの URL とタイトルを基本とする
- API サーバー URL はユーザーがポップアップで変更でき、ローカル保存される
- フォルダとタグは API 側の既存データを参照して選択する
- タグは複数選択、フォルダは単一選択とする
- API 接続成功時は、現在タブを一度だけ自動保存しようとする
- 重複 URL の場合はエラー表示ではなく既存ブックマークの読込へフォールバックする
- 削除は URL ベースで行い、保存済みブックマークがなければ失敗扱いにする
- API 接続失敗時はステータス表示で明示する
